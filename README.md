# Broke Buster

A command-line expense tracker built with Python. Lets you create spending 
categories and track how much you have spent on each one over time.

## What it does

You create categories like FOOD, TRANSPORT, RENT etc. and keep adding 
expenses to them. Each time you add an expense it accumulates on top of 
the previous amount so you can see your total spending per category.

## Features

- Add custom spending categories
- Add expenses to a category (accumulates over time)
- View all categories with their total spending
- Remove a category
- Reset all data

## How to run

Make sure you have Python installed, then run:

python main.py

## Files

- module1.py — contains the BrokeBuster class with all core logic
- main.py — entry point and menu handling
- brokebuster.txt — auto-generated file where your data is stored

## Built with

Python — file I/O, OOP, CLI input handling
